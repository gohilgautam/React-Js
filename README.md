## Projects

### Pixabay Clone
A high-fidelity clone of the Pixabay website built with React, TypeScript, and Tailwind CSS. This project aims to replicate the visual design and core functionality of Pixabay, offering a premium user experience for browsing and searching high-quality images.

#### Features
- Image Search: Seamless integration with the Pixabay API to search for thousands of free images.
- Advanced Filtering: Refine search results by image type, orientation, color, size, and sort order.
- Detailed Image View: A dedicated details page for each image with high-resolution preview, user profile, and statistics.
- Responsive Design: Fully responsive layout optimized for desktop, tablet, and mobile devices.
- Modern UI: Polished interface with smooth transitions, hover effects, and a clean aesthetic.
- Dark Mode Support: Built-in support for dark mode.

#### Tech Stack
- Frontend Framework: React (v18+)
- Language: TypeScript
- Build Tool: Vite
- Styling: Tailwind CSS
- Animations: Framer Motion
- Routing: React Router
- Icons: React Icons

#### Getting Started
1. Navigate to the Pixabay-Clone directory.
2. Install dependencies: `npm install`
3. Run the development server: `npm run dev`
4. Open `http://localhost:5173` in your browser.

For more details, see the [Pixabay-Clone README](Pixabay-Clone/README.md).
